# 🚗 Arduino Self-Driving Car

## 📌 Giới thiệu
Dự án xe tự điều khiển sử dụng Arduino.

## ⚙️ Công nghệ sử dụng
- Arduino Uno
- C++
- Cảm biến siêu âm HC-SR04

## 🚀 Chức năng
- Phát hiện vật cản
- Tự động tránh vật cản
- Di chuyển linh hoạt
